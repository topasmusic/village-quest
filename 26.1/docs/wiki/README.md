# Village Quest Wiki

This wiki documents the current shipped baseline of `Village Quest`.

Important note:
- the current published version is `1.20.0`
- this wiki now reflects the shipped `1.20.0` baseline and can continue forward from there

`Village Quest` is a village-centered progression mod for Minecraft `1.21.11`.
Its core loop is:
- help a village through grounded daily and weekly work
- earn trust and reputation
- unlock story arcs and long-term village projects
- earn relics and quiet magical tools
- take on optional roadside and outer-world work through the `Pilgrim`

## At a Glance

- `Questmaster` is the center of village-core progression.
- `Daily`, `Weekly`, `Story`, and `Special` all run through the `Questmaster`.
- `Story` unlocks after the first normal daily is completed.
- `Special` unlocks after the first real reputation gain.
- `Pilgrim` is not a second questmaster.
- `Pilgrim` handles optional road work, `Wayfinder` roadmarks, and later dangerous roadside rumors.
- `Watch Bell` is a late project that unlocks the `Roadside Watch` rumor lane after all four visible village-core story arcs are complete.
