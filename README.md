# Village Quest

This repository contains two maintained version lines of `Village Quest`.

- `1.21.11/` is the legacy line.
- `26.1/` is the current Minecraft `26.1` line.

Each folder builds on its own. Shared gameplay fixes can be ported between them, but version-specific code should stay in its own line because mappings, APIs, Java targets, and client hooks differ.

For local run and build commands, see [TEST_COMMANDS.md](TEST_COMMANDS.md).
